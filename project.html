Dungeon Hero Game

Used: C++
11/27/2023 - 12/04/2023

Dungeon Hero is a game created using OSU's FEH Proteus Simulator. It was submitted as the final project of ENGR 1281.01H, and it was created by Guy Levi (a fellow student) and myself. I wrote the code for the game while Guy created the graphics. The objective of the game is to escape the dungeon, all while killing enemies and collecting coins from chests. Players are able to upgrade the speed and power of their character using coins. The game is divided into levels and includes an infinite number of levels, providing hours of fun. The game is written in C++ and can be compiled following the instructions on the FEH Proteus website. You can download the game yourself using the link above!

 

This was the first major programming project that I had the pleasure of working on. I became a master of call-by-reference and all things related to pointers. I also learned how to effectively manage a codebase with a team, which is a skill I would not have considered to hone if not for this project.

Chinese Stock Research Paper

08/01/2021 - 08/31/2022

This is a research project I headed that investigated a possible correlation between the price of Chinese ADRs and various economic policies implemented by the Chinese Government. I analyzed the price movement of 53 different stocks after 31 key economic policies were passed in China. The final research paper ended up being 88 pages and concluded that there was no major correlation between the policies studied and the price movement of Chinese ADRs. The paper was published in the Journal of Student Research and has been downloaded nearly 100 times. It can be accessed from the link above.

 

This project was a great way to hone my technical writing skills. I also learned a great deal about collecting and analyzing data in an efficient manner. I became closely acquanted with SEC documents while hunting down obscure Chinese companies.
